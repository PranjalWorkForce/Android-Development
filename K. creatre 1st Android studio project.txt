Here’s a **clear, step-by-step guide** to create your **first Android project** using Android Studio.
-

# 🚀 Create Your First Android Studio Project

## ✅ 1. Install Android Studio

* Download from the official website (if not installed)
* Install with:

  * Android SDK
  * Emulator (AVD)
  * Required tools

---

## ▶️ 2. Launch Android Studio

* Open Android Studio
* Click **“Start a new Android Studio project”**

---

## 📱 3. Choose Project Template

You’ll see several templates:

* **Empty Activity** ✅ (Recommended for beginners)
* Basic Activity
* Navigation Drawer, etc.

👉 Select **Empty Activity** → Click **Next**

---

## 🧾 4. Configure Your Project

Fill in the details:

* **Name**: MyFirstApp
* **Package name**: com.example.myfirstapp
* **Save location**: Choose your folder
* **Language**:

  * Kotlin ✅ (recommended)
  * Java
* **Minimum SDK**:

  * API 21 (Android 5.0) is a safe choice

👉 Click **Finish**

---

## ⏳ 5. Wait for Gradle Build

* Android Studio will:

  * Download dependencies
  * Set up Gradle
* This may take a few minutes (first time only)

---

## 🧩 6. Understand Project Structure

Your project contains important folders:

### 📁 app/java/

* Your source code (Kotlin/Java)

### 📁 app/res/

* UI and resources:

  * layout → XML UI files
  * drawable → images
  * values → strings, colors

### 📄 AndroidManifest.xml

* App configuration (permissions, activities)

---

## 🖼️ 7. Open Main Activity

Go to:

```
app → java → MainActivity.kt
```

Example:

```kotlin
class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}
```

---

## 🎨 8. Design UI Layout

Open:

```
res → layout → activity_main.xml
```

Example UI:

```xml
<TextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello World!"
    android:layout_gravity="center"/>
```

---

## ▶️ 9. Run Your App

### Option A: Emulator

* Click **AVD Manager**
* Create a virtual device
* Click ▶️ Run

### Option B: Real Device

* Enable **USB Debugging**
* Connect phone
* Click ▶️ Run

---

## 🎉 Output

You’ll see:

```
Hello World!
```

on your screen 🎉

---

## 🔄 What Happens Behind the Scenes?

When you run the app:

* Code is compiled → `.dex`
* Resources are processed
* Everything packaged → `.apk`
* Installed on emulator/device

---

## 💡 Tips for Beginners

* Start with **Kotlin** (modern Android language)
* Learn:

  * Activities
  * Layouts (XML)
  * Intents
* Use Logcat for debugging

---

## 🧠 Next Steps

After your first app, try:

* Adding a **Button**
* Handling **click events**
* Navigating between screens
